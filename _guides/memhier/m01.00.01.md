You're probably in Dr. Peterson's class or Dr. Jantz's class looking for a **more-detailed writeup** for building this simulator.

In this guide, I'll preface by talking about the memory hierarchy and how accesses work, then I'll give some pointers for how to build this simulator, notably how each data structure behaves and how the simulation statistics are counted.
